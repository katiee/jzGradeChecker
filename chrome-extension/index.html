jzGradeChecker Chrome 扩展安装

您可以选择以下任意一种方式安装 jzGradeChecker Chrome 扩展。想了解扩展的更多信息请戳这里。

在 Chrome 网上应用店安装

点这里去 Chrome 网上应用店安装。

建议非 Chrome 用户使用下一种方法安装。

正常情况下 Chrome 会自动更新扩展。如果不能自动更新，只需直接覆盖安装新版本即可。

Pros: 没有“并非来自 Chrome 网上应用店”的提示，可自动同步扩展本身（考生数据目前不同步）。

Cons: 没有应用链接功能，可以快速在新标签页打开成绩查询界面。（Webstore 版本由于安全限制，无法实现此功能。）

在官方安装

在这里下载（版本 0.5，更新于 2013/5/12）。

由于 Chrome 的信任域问题，安装时，您很有可能需要打开 chrome://extensions 页面，把下载好的扩展文件拖入页面中，即可正常安装。

正常情况下 Chrome 会自动更新扩展。如果不能自动更新，只需直接覆盖安装新版本即可。

Pros: 提供应用链接，可以快速在新标签页打开成绩查询界面。（Webstore 版本由于安全限制，无法实现此功能。）

Cons: 非 Chrome 网上应用店安装的扩展目前无法跟随 Chrome 自动同步到其他电脑上。如果你有强迫症，看着“并非来自 Chrome 网上应用店”的提示会比较不爽吧。
